# Senderos REACT

Proyecto de un sistema web para la asignatura Sistemas Software Basados en Web del Máster Profesional de Ingeniería Informática de la Universidad de Granada.

Se va a desarrollar a lo largo de la tarea 8 por la alumna María Jesús López Salmerón.

Se va a intentar realizar una página similar a la página [TurGranada](https://www.turgranada.es/cosas-que-hacer/turismo-activo-y-de-naturaleza/excursiones-y-senderismo/) en donde se incluirán distintas rutas de senderismo y excursiones. Visualmente tiene las mismas características que el proyecto [Senderos de Django](https://github.com/mjls130598/Senderos).
